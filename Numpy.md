## 一、 Numpy数组：Array 

### 1.1 常用创建方式

1. **从列表转换 (`np.array`)** 

   - **场景**：手头有一个 Python list，想把它转成数组以便进行数学计算。

   - Python

     ```python
       arr = np.array([1, 2, 3])
     ```

2. **生成等差数列 (`np.arange`)** —— 类似 Python 的 range

   - **场景**：生成索引，或者比如“生成 0 到 100 岁的年龄段”。

   - Python

     ```python
       # 语法：np.arange(开始, 结束, 步长) (含头不含尾)
       arr = np.arange(0, 10, 2)  # [0, 2, 4, 6, 8]
     ```

3. **全 0 或全 1 矩阵 (`np.zeros`, `np.ones`)**

   - **场景**：先创建一个“空表”，后面再往里填数据。

   - Python

     ```python
       # 语法：传入形状元组 (行, 列)
       arr = np.zeros((3, 4)) # 生成 3行4列 的全0矩阵
     ```

4. **随机数 (`np.random`)**

   - **场景**：生成假数据来测试模型或代码逻辑（比如前文提到的 `randn`）。

   - Python

     ```python
       arr = np.random.randn(5, 2) # 5行2列的正态分布随机数
     ```



### 1.2 Numpy数组的属性

- `shape`：返回的是一个元组，说明这个数组的形状
- `ndim`：返回一个数值，说明数组的维度
- `dtype`：说明数组的数据类型

示例：

```python
import numpy as np

# 创建一个 2行3列 的数组
data = np.array([
    [10, 20, 30],
    [40, 50, 60]
])

print("1. ndim (维度):", data.ndim)
print("2. shape (形状):", data.shape)
print("3. dtype (类型):", data.dtype)
```

输出结果：

```
1. ndim (维度): 2
   (解析: 因为是二维数组，像个表格)

2. shape (形状): (2, 3)
   (解析: 意思是 2行，3列)

3. dtype (类型): int64
   (解析: 这里的数字 10, 20 都是整数，所以是 64位整数)
```



## 二、Numpy中的数据类型

NumPy 的数据类型 (`dtype`) 是它比 Python 原生列表（List）快几十倍甚至几百倍的内在原因

作为数据分析师，你只需要关注这四种：

#### ① 整数 (int) 

- **代码**：`int8`, `int16`, `int32`, `int64`
- **后缀的数字**：代表**“位 (bits)”**，也就是占用的内存大小。
  - **`int8`**：超小杯。只能存 -128 到 127。
  - **`int64`**：超大杯。能存天文数字。（现在的电脑默认都是这个）。
- **常见坑点**：如果你把一个很大的数（比如 1000）强行塞进 `int8`，数字会**溢出**（变成负数或乱码），而且不会报错！

#### ② 浮点数 (float) 

- **代码**：`float16`, `float32`, `float64`
- **默认值**：如果你创建一个带小数的数组 `[1.5, 2.3]`，NumPy 默认会给 `float64`（双精度），保证最精准。
- **关键知识点**：**`NaN` (缺失值) 属于浮点数！**
  - 如果你有一个整数列 `[1, 2, 3]`，把其中一个改成缺失值 `NaN`，整个数组会立刻被迫变成 `float`（`1.`，`2.`，`NaN`）。这是 Pandas 处理缺失值时的常见现象。

#### ③ 布尔值 (bool)

- **代码**：`bool`
- **内容**：只有 `True` (1) 和 `False` (0)。
- **用途**：主要用于**布尔索引**（做面具筛选数据）。

#### ④ 对象/字符串 (object / string) 

- **代码**：`S` (String), `U` (Unicode), `O` (Object)
- **含义**：
  - **`S` / `U`**：定长字符串。比如 `U10` 表示这个格子里最多存 10 个字的字符串。如果存了 11 个，第 11 个会被无情截断。
  - **`object`**：**大杂烩**。当你试图在 NumPy 里存不规则的字符串，或者混合类型（数字+字符串）时，NumPy 会放弃优化，直接退化成 Python 的列表模式。
  - **警报**：如果你看到你的数组 dtype 是 `object`，通常意味着计算效率会很低。



## 三、Numpy数组的矢量化运算

数组很重要，因为它使你不用编写循环即可对数据执行批量运算。NumPy用户称其为矢量化
`（vectorization）`。大小相等的数组之间的任何算术运算都会将运算应用到元素级：

```python
In [51]: arr = np.array([[1., 2., 3.], [4., 5., 6.]])
In [52]: arr
Out[52]:
array([[ 1., 2., 3.],
	   [ 4., 5., 6.]])

In [53]: arr * arr
Out[53]:
array([[ 1., 4., 9.],
	   [ 16., 25., 36.]])

In [54]: arr - arr
Out[54]:
array([[ 0., 0., 0.],
	   [ 0., 0., 0.]])
```

**广播机制 (Broadcasting)**

广播是 NumPy 的**“自动脑补”**功能。当两个数组形状不一样，但你非要让它们相加减时，NumPy 会尝试把小的那个**“拉伸”**成和大的那个一样大，然后再计算。

例：有 **3 个学生的成绩** `[80, 90, 100]`，给他们**每个人都加 5 分**。

```python
scores = np.array([80, 90, 100])
new_scores = scores + 5  # 数组 + 数字
```

**NumPy 的后台操作（广播）**： 它发现 `5` 只是个标量（数字），不够减。于是它自动把 `5` 变成了 `[5, 5, 5]`，然后和 `[80, 90, 100]` 对应位置相加。



## 四、索引和切片

### 4.1 基础的索引和切片

| **操作**   | **语法 (假设是二维 arr)** | **含义**                         |
| ---------- | ------------------------- | -------------------------------- |
| **找个点** | `arr[r, c]`               | 第 `r` 行，第 `c` 列             |
| **切行**   | `arr[r]`                  | 第 `r` 行的所有数据              |
| **切列**   | **`arr[:, c]`**           | 第 `c` 列的所有数据 (**最常用**) |
| **切块**   | `arr[r1:r2, c1:c2]`       | 只要这一块矩形区域               |
| **全选**   | `arr[:]` 或 `arr[:, :]`   | 整个数组                         |

推广到多维数组同理

**基础切片返回的是视图 (View)，不是副本。**

意思是，NumPy 为了省内存，切片出来的东西和原数组**共用一块内存**。你改了切片，原数组也会变！

例：

```python
arr = np.array([0, 10, 20, 30, 40])

# 1. 切出一块
slice_arr = arr[0:3]  # [0, 10, 20]

# 2. 修改切片里的第一个数
slice_arr[0] = 999

# 3. 检查原数组 —— 发现原数组也被改了！
print(arr)
# 结果: [999, 10, 20, 30, 40]
```

如果你想影响原数组，必须显式地用 `.copy()`： `slice_arr = arr[0:3].copy()`

### 4.2 布尔型索引（条件筛选）

用途：根据条件来筛选

原理：布尔索引分为两个步骤：**造面具** -> **戴面具**。

- **Step 1: 造面具 (生成 True/False 数组)** 当你写 `arr > 10` 时，NumPy 不会返回数字，而是返回一个和 `arr` 形状一模一样的**布尔数组**。
  - 原始数据：`arr = np.array([5,15,20])`
  - 条件 `arr > 10`
  - **生成的面具**：`[False, True, True]`
- **Step 2: 戴面具 (筛选数据)** 当你写 `arr[arr>10]` 时，NumPy 会把面具盖在原始数据上。
  - **True** 的位置：保留。
  - **False** 的位置：丢弃。
  - **最终结果**：`[15, 20]`

> 注意：Python关键字`and`和`or`在布尔型索引中无效。要使用`&`与`|`。



### 4.3 花式索引

**花式索引 (Fancy Indexing)**  允许你**“不按套路出牌”**，一次性拿走你想要的、不连续的、任意顺序的数据。索引位置放的不是数字，也不是冒号，而是一个数组（或列表）。

**基础语法：**

假设你有一个数组 `arr`：`[10, 20, 30, 40, 50, 60]`。

- 普通切片：`arr[0:3]` 只能拿前三个 `[10, 20, 30]`。

- 花式索引：拿第 0 个**、**第 3 个、第 5 个
  ```python
  import numpy as np
  arr = np.array([10, 20, 30, 40, 50, 60])
  
  # 注意：这里有两层括号，外层是索引操作，内层是一个列表
  print(arr[[0, 3, 5]])
  # 输出: [10, 40, 60]
  ```

  花式索引结果的顺序取决于传入的索引顺序：

  ```python
  # 我想要第 4 个，然后第 0 个
  print(arr[[4, 0]])
  # 输出: [50, 10]
  ```

  ### 副本 vs 视图

  这是面试常考题，也是实战中最容易出 Bug 的地方。

  - **普通切片 (`arr[0:5]`)**：返回的是**视图 (View)**。
    - 如果你修改了切片出来的结果，**原始数组也会变！**
  - **花式索引 (`arr[[0, 1]]`)**：返回的是**副本 (Copy)**。
    - NumPy 会把数据复制一份给你。你修改结果，**原始数组不会变**。

  **为什么？** 因为花式索引选出来的数据在内存里可能已经乱了（不连续），NumPy 为了保证效率和安全，干脆重新给你造一个新的内存块存这些数据。



**数组转置**

关键字：`.T`

```python
import numpy as np

# 创建一个 2行3列 的矩阵
arr = np.arange(6).reshape(2, 3)
# [[0, 1, 2],
#  [3, 4, 5]]

# 转置！
print(arr.T)
# 结果变成了 3行2列：
# [[0, 3],
#  [1, 4],
#  [2, 5]]
```



## 五、常用函数

####  统计聚合类 (计算总数、平均等)

这些函数通常都有一个 `axis` 参数（轴），决定是横着算还是竖着算。

- **`np.sum(arr)`** / **`arr.sum()`**: 求和
- **`np.mean(arr)`** / **`arr.mean()`**: 求平均值
- **`np.max(arr)`** / **`arr.max()`**: 求最大值
- **`np.std(arr)`**: 求标准差（波动程度）

> **关键点 `axis`**：
>
> - `axis=0`：**沿着竖直方向压缩**（算每一列的平均值/和）。-> **最常用**
> - `axis=1`：**沿着水平方向压缩**（算每一行的平均值/和）。

####  逻辑处理类 (清洗数据)

- **`np.where(condition, x, y)`**

  - **含义**：Excel 里的 `IF` 函数。

  - **语法**：`np.where(条件, 条件满足时填啥, 不满足时填啥)`

  - **例子**：把小于 60 分的标记为 0，其余标记为 1。

    ```python
    np.where(scores < 60, 0, 1)
    ```

- **`np.unique(arr)`**

  - **含义**：去除重复值，只保留唯一的（类似于 SQL 的 `DISTINCT`）。

- **`np.sort(arr)`**
  - **含义**：给数组排序。
